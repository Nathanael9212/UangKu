# 💰 UangKu

Aplikasi pencatatan keuangan pribadi berbasis React Native + Expo.

---

## Tentang Aplikasi

UangKu membantu kamu mencatat pemasukan dan pengeluaran sehari-hari dengan tampilan yang simpel dan mudah digunakan. Data tersimpan langsung di perangkat tanpa perlu akun atau koneksi internet.

---

## Fitur

- Catat pemasukan & pengeluaran dengan kategori
- Lihat ringkasan keuangan per Hari / Minggu / Bulan / Tahun
- Grafik distribusi keuangan
- Cari transaksi berdasarkan kata kunci, kategori, atau jumlah
- Set budget limit per kategori pengeluaran
- Export laporan ke Excel (.xlsx)

---

## Cara Menjalankan

```bash
npm install
npx expo start --clear
```

Tekan `W` untuk web, `A` untuk Android, `I` untuk iOS.

---

## Tech Stack

- React Native + Expo
- Expo Router
- Zustand + AsyncStorage
- date-fns
- xlsx
